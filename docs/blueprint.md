# **App Name**: Daily Script

## Core Features:

- Daily Scripture Feed: Display a new Bible verse daily from Firestore.
- Random Verse Generator: Generate a random Bible verse on user request.
- User Authentication: Allow users to sign in via Google, Apple, or anonymously.
- Favorite Verses: Enable users to save favorite verses to Firestore.
- Verse Sharing: Offer options to share verses on social media or copy the text.
- Daily Notifications: Send daily push notifications with the daily scripture.
- AI powered reflection: Use an LLM tool to add reflections or explanations of each verse

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke a sense of peace and spirituality.
- Background color: Very light blue (#E8EAF6) to maintain a light color scheme with minimal distraction.
- Accent color: Soft purple (#9575CD) for interactive elements.
- Body and headline font: 'Literata', a serif font to reflect literature.
- Use subtle icons such as scrolls, open books, or crosses.
- Maintain a clean and minimal layout, optimizing readability.
- Use subtle animations for verse transitions.
